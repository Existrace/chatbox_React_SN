# Petit chat de discussion en ligne

Fait en React avec Firebase (stockage des messages)

Fonctionnalités ajoutées : 
- Emoji
- Heure des messages

Fonctionnalités par encore réussies :
- Partage d'image

## Pour démarrer le projet :

yarn start 


Packages utilisés :

- Firebase
- Rebase
- react-router-dom
- moment
- emoji-mart
